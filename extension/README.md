# ClubHanger Capture — Arc / Chrome extension

Click the toolbar button on a Facebook group post to capture its text + photos
into ClubHanger (opens the capture page, which re-hosts images and saves a draft).

## Install in Arc (one time)

1. Open `arc://extensions` (type it in the address bar). *(Chrome: `chrome://extensions`)*
2. Turn on **Developer mode** (top-right toggle).
3. Click **Load unpacked**.
4. Select this folder:
   `/Users/zealoustiger/Dropbox/Code/aeromatch-next/extension`
5. The **ClubHanger Capture** plane icon appears in your extensions. Pin it to the
   toolbar so it's one click (Arc: extensions puzzle menu → pin).

## Use

1. Make sure you're signed into clubhanger.com as an admin in Arc.
2. Open a partnership post in your Facebook group — click into it so all photos load.
3. (Best results) Select the post text with your mouse.
4. Click the **ClubHanger Capture** toolbar button.
5. A ClubHanger tab opens and saves a draft → confirm it at
   [clubhanger.com/admin/review](https://clubhanger.com/admin/review).

## Updating

After any change to `background.js` or `manifest.json`, go back to `arc://extensions`
and click the **reload** (↻) icon on the ClubHanger Capture card.
