## `alerts` table shape
```
create table if not exists alerts (
  id          uuid        default gen_random_uuid() primary key,
  email       text        not null,
  context     text,          -- e.g. "Cessna 172" or "California"
  source_path text,          -- the page the signup came from
  status      text        not null default 'pending',  -- slice 2 = double-opt-in confirmation
  created_at  timestamptz default now(),
  unique(email, source_path)  -- same email+context twice = idempotent, not an error
);
alter table alerts enable row level security;
create policy "alerts_anyone_insert" on alerts for insert with check (true);
create policy "alerts_service_read"  on alerts for select using (auth.role() = 'service_role');
create index if not exists alerts_status_idx on alerts (status);
```
Strictly additive — no drop/alter-drop/delete/truncate, no existing table touched.
